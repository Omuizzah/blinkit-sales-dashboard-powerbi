# Blinkit Sales Dashboard Analysis

 ## Project Overview
This project presents an interactive dashboard built using Microsoft Power BI to analyze Blinkit grocery sales data.
It provides insights into sales performance, outlet location, and customer behavior.

## Objectives
- Analyze total and average sales
- Identify top-performing outlet locations
- Understand product distribution and ratings
- Compare outlet sizes and performance

## Tools Used
- Power BI
- Excel

# Dataset
The dataset contains grocery sales data including:
- Item type
- Outlet size
- Outlet location
- Sales
- Ratings

## Dashboard Features
- Total Sales: $1.20M
- Average Sales: $141
- Number of Items: 8523
- Average Rating: 3.9
- Sales by Item Type
- Outlet Location comparison
- Outlet Size distribution
- Sales trend over time
  
## Key Insights
- Tier 3 outlets generated the highest sales
- Supermarket Type 1 contributed the most revenue
- Low-fat products had strong sales performance
- Sales peaked around 2018–2019
